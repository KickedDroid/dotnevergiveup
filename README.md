# Dot NEVER GIVE UP

## Usage

Edit the config

```json
{
  "projectName": "MyProject",
  "dotnetVersion": "7.0",
  "packageName": "Microsoft.AspNetCore.Components",
  "packageVersion": "7.0.0",
  "buildConfiguration": "Release"
}
```

Add your `.cs` File with the name matching the name you listed in the config
